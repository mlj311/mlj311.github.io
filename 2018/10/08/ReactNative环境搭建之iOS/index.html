#

React Native-iOS环境搭建

一. React Native 主要依赖以下环境:

  • Mac OS X 操作系统
  • Xcode,推荐使用7.3.0或者更高系统
  • Node.js V6.2.1或者最新版本
  • Watchman和workflow
  • NVM

二.下载Xcode，这个就不多讲了，直接在App Store下载即可

三.安装Node.js，只要百度搜索Node.js，进入Node.js官网，直接下载

四.安装Homebrew, 打开 Homebrew官网, 语言选择简体中文, 按照文档步骤进行安装即可

五.打开终端依次输入命令：

  • 安装Homebrew:

  • 安装Yarn

  • 安装Flow
    ```brew install flow

    ```brew update && brew upgrade
  • 安装NVM
    ```git clone https://github.com/creationix/nvm
    ```cd nvm/
    ```source nvm.sh
    ```nvm
    ```nvm ls -remote N/A
    ```nvm ls

六.环境搭建完毕

七.创建项目

  • 在桌面创建一个文件夹HelloWorld
  • 打开终端进入此文件夹（cd）
  • 终端输入:
    react-native init HelloWorld
  • 在xcode中，使用快捷键cmd + R即可启动项目。基本的Xcode功能可以熟悉，比如模拟器的选择等。启动完成后，你会看到React-Packger和iOS模拟器，具体的效果如下，说明你创建项目成功了。
